Overview

WhisperSelf provides a simple REST API for speech-to-text transcription. Integrate it into your applications to add powerful voice transcription capabilities.

Base URL: http://localhost:3000

Authentication

All API requests require an API key in the request headers:

HTTP Header 
x-api-key: YOUR_API_KEY

Note: Contact your administrator to obtain an API key.

API Endpoints

GET /health

Check if the API is running.

Response

JSON Response 
{
  "success": true,
  "status": "ok"
}
GET /api/config

Get server configuration and limits. Requires authentication.

Response

JSON Response 
{
  "maxFileSizeMB": 100,
  "allowedTypes": [
    "audio/mpeg",
    "audio/wav",
    "audio/webm",
    "audio/mp4",
    "audio/ogg"
  ]
}
POST /transcribe

Transcribe audio or video files to text.

Request

Send a multipart/form-data request with the following fields:

Parameter Type Required Description
file File Yes Audio or video file to transcribe
language String No Language code (e.g., "en", "es", "fr"). Default: "auto"
task String No "transcribe" or "translate". Default: "transcribe"

Response

JSON Response 
{
  "success": true,
  "transcript": "Hello, this is a test transcription.",
  "language": "en",
  "duration": 5.23,
  "segments": [
    {
      "start": 0.0,
      "end": 5.23,
      "text": "Hello, this is a test transcription."
    }
  ]
}

Code Examples

cURL

Bash 
curl -X POST http://localhost:3000/transcribe \
  -H "x-api-key: YOUR_API_KEY" \
  -F "file=@/path/to/audio.mp3" \
  -F "language=auto" \
  -F "task=transcribe"

Python (requests)

Python 
import requests

url = "http://localhost:3000/transcribe"
headers = {"x-api-key": "YOUR_API_KEY"}
files = {"file": open("audio.mp3", "rb")}
data = {"language": "auto", "task": "transcribe"}

response = requests.post(url, headers=headers, files=files, data=data)
result = response.json()
print(result["transcript"])

JavaScript (Node.js)

JavaScript 
const FormData = require('form-data');
const fs = require('fs');
const axios = require('axios');

const form = new FormData();
form.append('file', fs.createReadStream('audio.mp3'));
form.append('language', 'auto');
form.append('task', 'transcribe');

axios.post('http://localhost:3000/transcribe', form, {
  headers: {
    ...form.getHeaders(),
    'x-api-key': 'YOUR_API_KEY'
  }
})
.then(response => console.log(response.data.transcript))
.catch(error => console.error(error));

Rate Limits

Transcription Endpoint: 10 requests per minute

If you exceed the rate limit, you'll receive a 429 (Too Many Requests) response. Wait for the rate limit window to reset before making additional requests.

Supported Formats

Audio

MP3, WAV, OGG, WebM, M4A, AAC

Video

MP4, WebM, AVI, MOV

Note: Video files are automatically processed to extract audio. Maximum file size is configurable (default: 100MB).

Error Handling

Status Code Error Description
400 Bad Request Invalid file format or missing required fields
401 Unauthorized Missing or invalid API key
413 Payload Too Large File size exceeds maximum limit
429 Too Many Requests Rate limit exceeded
500 Internal Server Error Transcription service error
